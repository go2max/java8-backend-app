Чтобы принять письмо можно было самостоятельно написать простой SMTP сервер, который бы принимал все команды от клиента и отвечал на них успешными сообщениями. Для кого-то установка настоящего SMTP сервера может показаться еще более простым вариантом, т.к. в задании не требуется его сложная конфигурация. Устанавливаем postfix: $ sudo apt-get install postfix В конфигурации "General type of mail configuration" выбираем "Internet Site". Доменное имя или "System mail name" пишем 2015.ppctf.net. Письмо отправляется на имя box, а пользователь с таким именем уже есть в системе, поэтому создавать его уже не нужно. Запускаем postfix сервер: $ sudo service postfix start Нажимаем "Submit", чтобы проверяющая система отправила нам письмо. Если все прошло гладко, то в файл /var/spool/mail/box запишется исходних принятого письмо. В конце письма находим архив-аттачмент с названием nothing.tar.gz, закодированный с помощью base64. Используя любой текстовый редактор, например vim или nano, вырезаем тело аттачмента и записываем в файл /home/box/nothing.tar.gz.base64 Декодируем base64: $ cd ~/ $ base64 -d nothing.tar.gz.base64 > nothing.tar.gz Распаковываем архив: $ tar xvzf nothing.tar.gz Из архива извлекается файл README, в котором находится флаг. Записываем флаг в файл /home/box/flag.txt: echo "STCTF#URTrueSMTPmaster#" > /home/box/flag.txt
