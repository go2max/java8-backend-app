(см. create/README.md для более подробной инструкции по созданию этого задания) В этом задании участникам предлагается воспользоваться забавной файловой системой: Pi file system, на что указывала ссылка на youtube из условия, а также написанное на футболке слово philipl -- это ник автора pifs на Гитхабе (https://github.com/philipl/pifs). Изначально дается архив с метаданными (metadata.zip), который нужно распаковать. Предполагается, что PiFS уже скачан и установлен (с https://github.com/philipl/pifs) Далее монтируем куда-нибудь нашу файловую систему, используя распакованный архив: $ ./src/pifs -o mdd=/metadata mnt/ Находим в mnt/ файл metadata.zip.b64, видим, что он закодирован в base64. Раскодируем: $ base64 -d metadata.zip.b64 > metadata.zip Опять получаем архив с метаданными (то есть перед нами было что-то вроде матрешки). Монтируем файловую систему еще раз, но уже с новыми метаданными, после чего в новой точке монтирования (mnt/ или что там еще) получаем файл flag.txt c флагом.
